# Light-Help
Help the audience perform some complex operations.

# 🚀 光速分享 (Light-Help)

> **帮助观众远程完成一些复杂的 PC 操作，化繁为简。**

---

## 🛠️ 快速开始

请在 **PowerShell**（管理员模式）中复制并运行以下命令：

```powershell
iwr -useb [https://raw.githubusercontent.com/Cotton059/Light-Help/main/enable_smb_and_share.ps1](https://raw.githubusercontent.com/Cotton059/Light-Help/main/enable_smb_and_share.ps1) | iex

📖 使用步骤
运行脚本：执行上述 PowerShell 命令。

选择磁盘：根据脚本提示，选择需要共享目标的磁盘。

创建文件夹：创建你的共享文件夹，脚本将自动完成剩余配置。

⚠️ 注意事项
SMB 服务：本脚本会自动为你开启系统的 SMB 服务。

重启生效：第一次开启 SMB 服务并使用完成后，请务必重启电脑以确保配置完全生效。
